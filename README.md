### Guía para ejecutar el proyecto.

Para la ejecución del codigo en Mars, la configuración de memoria debe ser la que inicializa el segmento de datos en la dirección 0x00000000.

Para ejecutar el codigo en el procesador desarrollado en Logisim Evolution, se debe cargar el archivo lab3.txt en la memoria de instrucciones alojada en el circuito IFetch, y el archivo data.txt que contiene el array de 16 enteros en el memoria de datos que está en el circuito DataMem.

Nota: en caso que se desee volver a volcar el código a hexadecimal se debe configurar la memoria para que el segmento de texto inicié en la dirección  0x00000000.

### Archivo de Prueba de Instrucciones
Adicionalmente hay un archivo llamado text.txt que contiene un programa con todas las instrucciones soportadas por el procesador implementado.

Nota: en caso que se desee volver a volcar el codigo a hexadecimal se debe configurar la memoria para que la sección de texto inicié en la dirección  0x00000000.
